About

FOSSEE, IIT Bombay is hosting an online Scilab Toolbox Hackathon to develop Scilab Toolboxes. So get started and utilize your lockdown time to participate in this hackathon and improve your coding skills.

What is Scilab Toolbox Hackathon?

Scilab Toolbox Hackathon is an online, nonstop "Scilab Toolbox Development" based on an established open source software, a competition initiated by the FOSSEE Project, IIT Bombay. It aims at inspiring minds across the globe to build outstanding toolboxes for Scilab.

Eligibility to participate

If you are a good software developer(C, R, Python etc,.) and know how to interface multiple software, you can quickly learn Scilab toolbox development using Spoken tutorials and participate. Register using the Google form and develop Scilab toolbox as an individual or as a team(max. 4 members) and win prizes. Optionally, you can seek guidance from a mentor.

Problem Statement

Build a Scilab Toolbox by making available a well established open source software callable from Scilab. This process can be broken down into the following steps.

  1. Selection of open source software:
    1. First, identify an open source software whose functionalities/features you would like to include in Scilab. This software can be of any domain such as sciences, any branch of engineering, statistics, etc.
    2. The above will be done by the team member who has the domain expertise.
    3. The code should be a well established popular software, which if made available as a Scilab Toolbox, will help many.
  2. Interfacing the chosen software with Scilab:
    1. This task has to be done by the team members who are comfortable with interfacing two or more software components with each other.
    2. This is someone who has some CS/IT expertise, even though they may be from application domains also.
  3. Verification and Testing of the toolbox:
    1. For verification, two or more number of examples should be demonstrated using the created toolbox.
    2. Answers given by the Toolbox should be identical to the one given by the underlying software in native mode.

What is the Scilab toolbox?

A Scilab Toolbox is a collection of functions that solve problems of a particular subject. For example, the FOSSEE Optimization Toolbox, FOT, is a collection of COIN-OR libraries that are written to solve Optimization problems. Creating a toolbox out of such popular and useful open source routines enables their usage from Scilab thereby contributing a great resource to Scilab users globally.

Rules and Regulations

  • The decision of the review committee and organizers in declaring the results will be final. No queries in this regard will be entertained.
  • The same person can not participate in more than one team.
  • There will be only one point of contact (the team leader) with the organizers.
  • Any participant found to be indulging in any form of malpractice will be immediately disqualified.
